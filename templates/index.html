Movies API

About

Welcome to the free Movies API! The data includes 4803 movies. This documentation should supply with you all the information you need to start making your GET HTTP requests. Feel free to use it in your demo projects, tutorials, or testing tasks.

Endpoints

  1. Base URL
    https://retrivedataset.herokuapp.com

  2. Fetch all movies
    https://retrivedataset.herokuapp.com/api/v1/resources/movies/all

  3. Fetch movies by title
    https://retrivedataset.herokuapp.com/api/v1/resources/movies/title/<title>
